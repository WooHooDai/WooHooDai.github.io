尝试切换到Debian Jessie系统下.做些记录,避免无用的折腾.

系统安装

  1. 缺失固件

    • 安装过程中会有提醒,最好提前下载,放入启动盘的firmware里.这样安装程序会为固件自动添加更新源
    • 或者安装好系统后直接下载deb安装

  2. 空间划分

    • /boot 独立(1GB)
    • 用LVM配置根目录,/home,/swap
    • /swap保持和内存大小一致,/home有个10~20GB就好,剩下的给根目录

  3. 引导问题

    • 系统安装完成后,使用 update-grub 命令,使得windows能被引导
    • 文件 \etc\default\grub 可配置默认启动系统,和等待时间

  4. 中文字体: sudo apt-get install xfonts-wqy 文泉驿点阵宋体

硬件问题

  1. 花屏问题

    • 如果是花屏导致无法进入系统,先参考这里
      • 在Grub界面按e编辑, NVIDIA显卡在quiet splash后加nomodeset
      • 进入系统后
        • sudo gedit /etc/default/grub
        • 将GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"改为GRUB_CMDLINE_LINUX_DEFAULT="quiet splash nomodeset"
        • sudo update-grub
      • nomodeset会关掉KMS从而禁用nouveau,导致屏幕亮度无法调节,所以应当接着安装nvidia私有驱动
      • 多显卡参考这里,下面是多显卡.单显卡参考这里
        • sudo apt-get install bumblebee-nvidia primus
        • sudo adduser $USER bumblebee
        • 重启
        • optirun --help查看一些设置
      • 值得参考的资料
      • KMS: Kernel mode setting
      • NVIDIA
      • Nouveau
      • nomodeset与blacklist禁用nouveau的区别(2楼)

  2. 声音

    • 只在第一次安装时出现没有声音的问题(且并非第一次进入系统出现,怀疑和某些软件冲突)
    • 解决的办法是:
      • alsamixer 里调整好各项设置
      • sudo alsactl init
      • 以root身份登录系统,没有该问题.以其他用户身份登录,则每次都需要运行上述命令.可以考虑加入开机启动

  3. 笔记本内置键盘:触摸板,键盘,键盘背光

    • 禁用触摸板
      • sudo modprobe -r psmouse 永久有效
      • 启用触摸板 sudo modprobe psmouse

    • 禁用笔记本内置键盘

      • 使用xinput

  4. 蓝牙

    • dmesg | grep Bluetooth 查看和蓝牙相关的内核运行信息,得到如下内容

系统使用

  1. 修改apt更新源

    • 在文件 /etc/apt/sources.list 中修改
    • 刚安装完系统,需要注释掉 deb cdrom 那一行

  2. 将用户设置为 sudoer

    • 安装sudo

      1
2

      su    # 切换到root权限
apt-get install sudo    # 安装sudo

    • 添加用户到 /etc/sudoers

      1
2
3

      su    # 切换到root权限
visudo    # 更好的编辑该文件方式
# 在 `root ALL=(ALL:ALL) ALL` 一行下添加 `woohoodai ALL=(ALL:ALL) NOPASSWD:ALL`

  3. 自启动

    • 在 /etc/rc.local 中添加命令行的方式不生效,还需要再研究.另外这也不是systemd的自启动方式

    • 使用systemd开机启动: 以我的需求为例

      • 参考阮一峰老师的Systemd 入门教程：实战篇

      • 准备一个需要开机自启动的脚本,这里是 /usr/local/bin/myautostart.sh ,脚本内容如下

软件安装

通用说明

  1. 通过 apt-get 安装,但下载失败的程序,可以将手动下载好的文件放入 /var/cache/apt/archives ,然后带参数 --fix-missing 重新执行 install

  2. 自己下载的deb文件,可以通过 sudo dpkg -i xxx.deb 来安装

  3. 直接可用的程序,可以添加到 附件-主菜单 ,方便搜索和使用

个性化定制

  1. 设置图标大小

    • dconf中 org-gnome-desktop-nautilus-icon-view

  2. 桌面样式

    • 备注:使用火狐浏览器,可通过网站自动安装,节约时间
    • 在 系统工具-优化工具-扩展 中进行设置
    • Applications menu #在任务栏显示应用程序,类似于win的开始菜单
    • Friippery move clock #将时间移到右侧
    • #Hide top bar #隐藏任务栏
    • Native window placement #更好的展示活动程序
    • Openweather #天气情况,很好用
    • Places status indicator #在任务栏显示用户目录文件夹入口
    • Taskbar #神器,可替代Window list
    • #Window list #在屏幕底部显示开启的程序,类似于win的任务栏
    • EasyScreenCast # 录屏录音
    • Mmod panel # 将topbar放到屏幕底部
    • apt-get install maximus 去标题栏,设置下开机启动

  3. 字体

    • Infinality 字体渲染 (梳理自这里)

      • 安装
        • 添加源: /etc/ap/sources.list里添加 > deb http://ppa.launchpad.net/no1wantdthisname/ppa/ubuntu trusty main > deb-src http://ppa.launchpad.net/no1wantdthisname/ppa/ubuntu trusty main
        • 添加PPA的PGP公钥: sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys E985B27B
        • 更新: sudo apt-get update
        • 安装: sudo apt-get install fontconfig-infinality
        • 使用
        • 直接选择字体风格: sudo bash /etc/fonts/infinality/infctl.sh setstyle
        • 或者编辑文件/etc/profile.d/infinality-settings.sh中的USE_STYLE字段

    • 字体可以在gnome-tweak-tool里设置

    • Iosevka 字体

      • 下载Iosevka字体
      • 将字体文件放到 ~/.local/share/fonts,执行 sudo fc-cache
      • 在 系统工具-优化工具-字体 中使用该字体,将微调设为Full,抗锯齿设为Rgba

    • Ubuntu 字体

      • 在这里下载
      • 安装见[软件安装-通用说明]

  4. 快捷键

    • 终端:
      • Ctrl+T 新标签页打开新终端
      • Super+T 新窗口打开新终端
      • Shift+Ctrl+W 关闭当前终端
      • Shift+Ctrl+C 复制
      • Shift+Ctrl+V 粘贴
    • 全局: 全部设置-键盘-快捷键
      • Super+A 显示活动程序概览
      • Super+Space 显示消息栏
      • Super+D 隐藏所有正常窗口(显示桌面) 在[导航]里
      • Ctrl+Alt+A 选取截屏,保存在picture 在[搜索]里
      • Ctrl+Alt+C 选取截屏,复制到剪贴板里
      • Ctrl+Super+T 新窗口打开新终端 gnome-terminal # 配合nautilus,使得在任意位置都能打开一个终端
      • #Super+Esc suspend to disk echo "disk" | sudo tee /sys/power/state # 通过快捷键或者建立主菜单经常无响应.此外再开机后会卡顿很久,弃用
      • Ctrl+Shift+Delete 打开任务管理器(gnome-system-monitor)
      • Super+Backspace 静音
      • Super+= 增大音量
      • Super+- 减小音量
      • #Ctrl+Alt+A 截屏(gnome-terminal –geometry=0x0+9999+9999 –hide-menubar -e “/home/woohoodai/nutstore/bin/screenshot.sh”)
      • Super+Home 打开工作空间
      • Ctrl+Alt+Delete 打开任务管理器(gnome-system-monitor)
    • 预设
      • 可在~/.config中对应文件夹的accels文件中修改快捷键
        • 修改之前彻底关闭目标程序(可以通过gnome-system-monitor)
        • 记得将语句前的分号去掉
      • Super+T 在当前目录打开终端
      • Ctrl+D(进入对应的文件夹后) Nautilus侧边栏添加书签
