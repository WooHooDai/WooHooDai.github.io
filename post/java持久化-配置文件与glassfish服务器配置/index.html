涉及persistence.xml和glassfish服务器的配置

persistence.xml

  1. 配置文件放到应用资源文件夹’resources’下的META-INF文件夹下
  2. 配置内容如下:

<?xml version="1.0" encoding="UTF-8"?>
<persistence version="2.1" 
             xmlns="http://xmlns.jcp.org/xml/ns/persistence" 
             xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" 
             xsi:schemaLocation="http://xmlns.jcp.org/xml/ns/persistence http://xmlns.jcp.org/xml/ns/persistence/persistence_2_1.xsd">
    <persistence-unit name="dukebookstore" transaction-type="JTA">
        <jta-data-source>jdbc/dukebookstore</jta-data-source>
        <properties>
            <property name="javax.persistence.schema-generation.database.action" value="drop-and-create" />
            <property name="eclipselink.ddl-generation.output-mode" value="both" />
        </properties>
    </persistence-unit>
</persistence>
  • 其中<jta-data-source>标签中的资源需要在服务器中配置
  • 其中EclipseLink用于自动创建数据库模式

GlassFish服务器配置

需要注意的点

  1. GlassFish需要先建一个连接池, 再建JNDI资源
  2. JNDI资源名用于数据库时格式一般是jdbc/名字, 其中名字的部分以小写字母开头
  3. 建立连接池时, 需提前把数据库驱动放到[domainname]/lib下, 并重启服务器. 更多信息参考这里
  4. 在服务器中配置数据库驱动详见这里

使用MySQL的具体配置

  1. 将驱动程序mysql-connector-java-5.1.35-bin.jar放到glassfish4/glassfish/domains/domain1/lib下, 并重启服务器
  2. 在服务器管理界面中的Resources-JDBC-JDBC Connection Pools中新建一个连接池
    • Pool Name: MySQLPool
    • Resource Type: javax.sql.ConnectionPoolDataSource
    • Datasource Classname: com.mysql.jdbc.jdbc2.optional.MysqlDataSource
    • 在Additional Properties中配置如下数据库属性
      • DatabaseName: forjavaee
      • ServerName: localhost
      • User: woohoodai
      • PortNumber: 3306
      • Password:
  3. 在服务器管理界面中的JDBC Resources里建立JNDI资源
    • JNDI Name: jdbc/aname
    • Pool Name: MySQLPool
  4. 在persistence.xml中配置好持久化单元即可
