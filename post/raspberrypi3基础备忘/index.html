折腾树莓派的一些记录

选购SD卡

  • 树莓派并不支持所有型号的SD或者TF卡，兼容列表参考这里
  • 一般来说闪迪16GB系列的兼容度最高，都没啥问题
  • 最后买的SanDisk microSDHC™ UHS-I 16GB（说是红卡比白卡快，倒是贵了一点，没觉得快）

安装系统

  1. NOOBS

    • NOOBS是官方提供的树莓派系统的安装软件
    • 用NOOBS安装系统需要你有一个显示器和视频线
    • 具体安装需求和过程见官方教程
    • NOOBS下载地址在这里

  2. 直接烧录系统

    • 在这里下载SD Card Formatter，格式化一下SD卡。如果格式化后，发现显示的剩余空间与SD卡实际大小不符，请在软件的[选项设置]里，将[逻辑大小调整]设为[开启(ON)]
    • 下载一个系统。这里下载官方的RASPBIAN系统。也可以在这里根据用途选取其他的系统。
    • 在这里下载Win32 Disk Imager。安装好以后用管理员权限打开。读入刚才下载好的系统(.img)，点击[Write]将系统写入SD卡

通过SSH控制树莓派 （无显示器，无路由器）

  1. 用一根网线连接树莓派和一台无线上网的电脑，给树莓派通电
  2. [打开网络和共享中心]——[更改适配器设置]——右键[WLAN][属性]——[共享]标签页——勾选[允许其他网络用户通过此计算机的Internet连接来连接]
  3. 打开cmd.exe，输入 arp -a，在 接口：192.168.137.1 下找到树莓派的IP地址
  4. 在这里下载PuTTY。在[Host Name(or IP address)]下的输入框里输入 pi@你的树莓派IP地址，点击下面的[Open]，输入默认密码 raspberry即可成功登录树莓派

设置树莓派

  1. 在SSH中输入 sudo raspi-config 进入设置
  2. [Expand Filesystem] 是要点的，将整张SD卡的空间供系统使用
  3. [Change User Password] 更改树莓派的登录密码
  4. [Internationalisation Options] 将系统设置为中文。设置成中文后需要下载中文字体，可以通过命令 sudo apt-get install ttf-wqy-microhei ttf-wqy-zenhei xfonts-wqy
  5. 其他设置参考官方文档

更换apt源

  1. 官方apt较慢，官方提供了一个镜像源列表。推荐清华大学TUNA镜像源和阿里云开源镜像站

  2. 设置方法如下：

    • sudo nano /etc/apt/sources.list

    • 将文件中相应内容更换为镜像源，如更换为清华大学镜像则更换为如下内容

      1
2

      deb http://mirrors.tuna.tsinghua.edu.cn/raspbian/raspbian/ jessie main non-free contrib  
deb-src http://mirrors.tuna.tsinghua.edu.cn/raspbian/raspbian/ jessie main non-free contrib

    • ctrl+o,enter,ctrl+x

    • sudo apt-get update

为树莓派设置固定IP

  • sudo nano /etc/network/interfaces

  • 把 auto lo 和 allow-hotplug wlan0 之间的部分改为如下内容，其中 192.168.137.23 即为树莓派IP地址

    1
2
3
4
5
6

    iface lo inet loopback  
iface eth0 inet static  
adress 192.168.137.23  
netmask 255.255.255.0  
gateway 192.168.137.1  
dns-nameservers 8.8.8.8

  • ctrl+o,enter,ctrl+x

SSH免密码登录

  1. 可参考官方教程

  2. 在电脑上生成rsa: ssh-keygen -t rsa

  3. 将公钥(.pub)拷到pi上，位置为[/home/pi/.ssh]。可以考虑使用FileZilla

  4. 将公钥写入pi的authorized_keys，在SSH里输入: cat ~/.ssh/id_rsa.pub | ssh <USERNAME>@<IP-ADDRESS> 'cat >> .ssh/authorized_keys'

  5. 在电脑上：

    • 用PuTTY的话，需要先用PuTTYgen将私钥转换成ppk格式，然后在PuTTY里的[Connection]-[SSH]-[Auth]-[Private file for authentication]里加载这个ppk文件
    • 直接用控制台的话，需要用ssh-add加入私钥: ssh-add 你的私钥的文件路径

VNC：远程桌面

  1. 可以通过VNC，在各种设备上控制树莓派。不过树莓派和控制设备需要处于同一网络下

  2. 通过SSH，在pi上安装和启动vnc

    1
2
3

    sudo apt-get install tightvncserver`  #安装vncserver  
vncpasswd`  #设置密码  
vncserver :1 -geometry 800x600`  #启动

  3. 在控制设备上下载对应的VNC

  4. 在控制设备上的VNC中输入 树莓派的IP地址:启动的图形界面编号 如 192.168.137.23:1，点击[Connect]，输入刚才设置的密码后即可登录

  5. 使树莓派开机自动启动vnc服务

# Required-Stop: $remote_fs $syslog

#http://www.penguintutor.com/linux/tightvnc

eval cd ~$VNCUSER  
case "$1" in  
start)  
   su $VNCUSER -c '/usr/bin/tightvncserver :1  -geometry 1024x640 -depth 16 -pixelformat rgb565'  
   echo "Starting TightVNC server for $VNCUSER "  
   ;;  
stop)  
   pkill Xtightvnc  
   echo "Tightvncserver stopped"  
   ;;  
*)  
   echo "Usage: /etc/init.d/tightvncserver {start|stop}"  
   exit 1  
   ;;  
esac  
exit 0

    • sudo update-rc.d tightvncserver defaults

安装显示器

  1. 因为担心命令行搞不掂，于是顺手买了一个小屏幕没卵用。总之这种直插的显示器需要安装驱动程序，找店老板要就好。下面的只是自留备忘
  2. 下载驱动程序，拷贝到pi上

  3. 进行如下操作

    tar -xzvf LCD_show_v6.tar.gz #解压缩
    cd LCD_show_v6 #进入文件夹
    sudo apt-get update #升级更新系统
    sudo ./LCD_backup #备份数据
    sudo ./LCD35_v #安装驱动

  4. 恢复系统: sudo ./LCD_restore

  5. 更新系统，需要如下操作，否则需要重新安装驱动

    sudo apt-mark hold raspberrypi-bootloader
    sudo apt-get update
    sudo apt-get upgrade
